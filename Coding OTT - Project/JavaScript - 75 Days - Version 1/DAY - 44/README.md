## JS - V1 - DAY: 44

### 📌 Task:
Ye task **Day 42 - 43** ke same hi hai, bas difference itna hai ki isme user ko **multiple images/files select karne ka option** dena hai.

---

### ✅ Requirements:

- User ek saath **multiple images/files select** kar sake.
- Agar user ne **koi bhi image upload nahi ki hai**, toh screen par **"No Images Found"** ka message show karo **icon ke saath**.
- Page ke **bilkul bottom** me ek **Upload Button** do jisse user multiple photos select kar sake.
- Select karne ke baad sabhi images ko website par display karo.

---

### 🖼️ Image Display Features:

Har uploaded image/file ke saath ye cheezein show honi chahiye:

- **Delete icon**
- **View icon**
- **Image Name** (icon ke saath)
- **Image Size** (icon ke saath)

⚠️ Note: Agar image name ya content zyada bada ho toh **overflow content ko properly control** karo (truncate / wrap / scroll).

---

### 🗑️ Delete Functionality:

- Jab user **delete icon** par click kare, toh ek **custom confirmation popup** open hona chahiye.
- Popup me ye cheezein honi chahiye:

  - **Title**
  - **Description**
  - **Cancel Button**
  - **Confirm Button**

- Confirm karne ke baad selected image/file delete ho jaye.

---

### 👁️ View Functionality:

- Jab user **view icon** par click kare, toh ek **full screen popup/modal** open ho.
- Us modal me image **full screen** par display honi chahiye.
- Modal ko close karne ke liye ek **close icon** bhi hona chahiye.
