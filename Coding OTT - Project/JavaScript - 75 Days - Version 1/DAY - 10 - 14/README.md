# JS V1 - Day 10 - 14

## Task
Ek aisa project banaiye jisme user **Registration** kar sake aur user ki details ko **Array aur Object** ke andar store kiya jaye.  

Agar user dobara same data fill kare, toh **new object create karne ke bajaye existing object ko update** kiya jaye.  

---

## Registration Form
- User se details input lena hai (jaise: Name, Email, Password, etc.)
- Data ko **Array ke andar objects ke form me store** karna hai.
- Agar user same email se dobara register kare toh uska object **update** hona chahiye.

---

## Login Form
Ek **Login Form** bhi banana hai jisme:
- User **Email aur Password** enter kare.
- Enter kiya gaya data **stored object/array data se match** kare.
- Match hone par login successful show kare, warna failed message show kare.

---

## Popup Dialog + Animation
- **Animate.css CDN** use karna hai.
- Custom **popup dialog/message box** banana hai.
- Popup message different conditions par show hona chahiye:

✅ Registration Successful  
✅ Login Successful  
❌ Login Failed / Invalid Credentials  

Popup me animation apply karne ke liye **animate.css classes dynamically add/remove** karni hongi.
