# JS V1 - Day 9

## Task 1
Ek aisa project banaiye jisme ek card ke andar **Title** aur **Description** ho, aur saath hi ek **Color Picker Input** diya ho.  

User jis element par click kare (chahe **Title** ho ya **Description**), aur phir color select kare, toh **sirf usi selected element ka color change hona chahiye**.  

✅ **Default behavior:** By default **Title** ka color change hona chahiye.  
✅ Color change hone ke saath-saath **selected color ka HEX code bhi show hona chahiye**.
