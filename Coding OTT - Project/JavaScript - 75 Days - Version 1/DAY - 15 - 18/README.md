# JS V1 - Day 15 - 18

## Task
Ek aisa project banaiye jisme ek **Calculator** ho aur uski saari calculations **JavaScript** se perform ho.  

User jab bhi calculation kare, toh result screen par show hona chahiye.  

### Extra Condition
Result show karte time yeh bhi check karna hai ki output me jo number aaya hai woh:

- **Even (Even Number)** hai  
ya  
- **Odd (Odd Number)** hai  

Phir uske according ek **text message** bhi display karna hai.

✅ Example:  
- Output: `10` → Message: **"This is an Even Number"**  
- Output: `7` → Message: **"This is an Odd Number"**
