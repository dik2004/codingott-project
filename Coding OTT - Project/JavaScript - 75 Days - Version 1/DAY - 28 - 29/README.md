# JS V1 - Day 28 - 29

## Task
Ek aisa project banaiye jisme **Headings, Paragraph, OL, UL, LI** jaise HTML elements ko **JavaScript se dynamically create** kiya jaye.

---

## Requirements
- Page par different buttons hone chahiye (Example: Add Heading, Add Paragraph, Add OL, Add UL, Add LI).
- User jab kisi button par click kare, toh us button ke according specific element dynamically create ho.

---

## Elements to Create Dynamically
Button click par dynamically create karna hai:

- **Heading (h1 to h6)**
- **Paragraph (p)**
- **Ordered List (ol)**
- **Unordered List (ul)**
- **List Items (li)**

---

## Working
- Har button click par new element create hoke page me add hona chahiye.
- Saare elements JavaScript ke through create hone chahiye using DOM methods like:
  - `createElement()`
  - `appendChild()`
  - `innerText / innerHTML`

---

## Important Instruction
Is project me saare HTML tags ko **dynamically JavaScript ke through create aur display** karna hai.
