## JS - V1 - DAY: 51

### 📌 Task:
Ek aisa project banao jisme ek **Login Form** ho aur usme **Remember Me** functionality bhi properly work kare.

---

### ✅ Login Form Requirements:

Login form me ye fields hone chahiye:

- **Email Input**
- **Password Input**
  - Password field me **show/hide password icon** hona chahiye

---

### 👁️ Show / Hide Password Feature:

- Password input ke andar ek icon ho (eye icon)
- Icon par click karne par:
  - password show ho jaye
  - aur icon change ho jaye
- Dobara click karne par:
  - password hide ho jaye
  - aur icon wapas change ho jaye

---

### ☑️ Remember Me Feature:

- Password field ke niche ek **Remember Me checkbox** hona chahiye.
- Jab user checkbox select kare aur login kare, toh:
  - user ka **email aur password Local Storage** me save ho jaye.

---

### 🔄 Auto Fill Condition (Local Storage):

- Website refresh/open hone par Local Storage check karo:
  - agar user ne pehle **Remember Me** select kiya tha,
    toh login form me **email aur password auto fill** ho jaye.
  - agar Remember Me select nahi tha,
    toh inputs empty rahne chahiye.

---

### 🎉 Login Success Popup:

- Login button click karne ke baad ek **custom popup/dialog** show karo.
- Popup me:

  - **Success Title**
  - **Success Message**
  - Close/Ok button

- Popup professional UI me hona chahiye.
