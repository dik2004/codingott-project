## JS - V1 - DAY: 47 - 50

### 📌 Task:
Ek aisa project banao jisme user apne device se **audio/music upload karke play** kar sake.

---

### ✅ Initial Screen (First Time Open)

- Jab user website open kare aur koi music select na ho, tab:
  - **"No Music Found"** message show karo (icon ke saath)
  - ek button show karo: **"Select Music"**

---

### 🎵 Audio Select Karne Ke Baad

- Jaise hi user music file upload kare:
  - Audio player UI show ho jaye
  - button ka text change ho jaye: **"Choose Another Music"**

---

### 🎶 Audio Player Features

Audio player me ye sab features hone chahiye:

- **Play / Pause button**
- **Forward button**
- **Backward button**
- **Mute / Unmute button**
- **Loop button**
- **Progress bar** (seek functionality ke saath)

---

### ⏱️ Time & Duration Display

Audio player me ye info show honi chahiye:

- **Current Time** (audio kitna play ho chuka hai)
- **Total Duration** (audio ka total time)
- Progress bar ke through audio ka progress properly track ho

---

### 📄 Audio File Details

Uploaded music file ka:

- **Music Name** (icon ke saath)
- **Music Size** (icon ke saath)

ye bhi show karna hai.

---

### 🔄 Proper Tracking & Icon Changes

System properly track kare ki:

- audio **play** ho raha hai
- audio **pause** hai
- audio **end** ho gaya hai
- audio **loop mode** on/off hai
- audio **mute/unmute** hai

⚡ User ke actions ke according icons dynamically change hone chahiye.

Example:
- Play click -> Pause icon show ho
- Pause click -> Play icon show ho
- Mute click -> Unmute icon show ho
- Loop on -> loop active icon show ho
