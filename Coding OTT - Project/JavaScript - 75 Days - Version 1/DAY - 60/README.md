## JS - V1 - DAY: 60

### 📌 Task:
Ek aisa project banao jisme **Promise se data fetch** kiya ja raha ho aur usme **try-catch** ka use ho.

---

### ✅ Requirements:

- Ek function banao jo **Promise return** kare.
- Promise ke andar `setTimeout()` use karo (delay create karne ke liye).
- Promise condition ke basis par:
  - `resolve()` kare ya
  - `reject()` kare

---

### ⚡ Async / Await Condition:

- Ek async function banao jisme:
  - `await` ka use karke promise ka data properly wait karo
  - data aane se pehle UI par result show nahi hona chahiye

---

### 🛡️ Try Catch Condition:

- `await` ke saath **try-catch** ka use compulsory hai.
- Agar promise resolve ho:
  - success message show karo
- Agar promise reject ho:
  - error message show karo

---

### 🎯 Output Show:

- Result ko UI par show karna hai (success ya error message).
- `console.log()` ke saath saath webpage par bhi output visible hona chahiye.

---

### ⚠️ Note:

- Project ka goal ye samajhna hai ki **async-await** promise ke complete hone ka wait karta hai,
  aur **try-catch** error handling ke liye use hota hai.
