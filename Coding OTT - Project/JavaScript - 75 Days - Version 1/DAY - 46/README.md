## JS - V1 - DAY: 46

### 📌 Task:
Ek aisa project banao jisme user **PDF file ko read/view** kar sake **iframe** ka use karke.

---

### ✅ Requirements:

- User ko ek **PDF file select/upload** karne ka option dena hai.
- PDF ko upload karne ke baad usko **iframe ke andar display** karna hai (PDF preview).

---

### 📄 No PDF Condition:

- Agar user ne **koi bhi PDF file select nahi ki hai**, toh screen par:
  - **"No PDF Found"** message show karo
  - message ke saath ek **icon** bhi hona chahiye

---

### 📌 PDF Upload ke baad ye details show karo:

PDF file upload hone ke baad ye information show honi chahiye:

- **PDF Name**
- **PDF Size**
- **Last Modified Date**

---

### 🔄 Button Text Condition:

- Default button text: **"Select File"**
- Jab user PDF upload kar de, toh button ka text change karke:
  - **"Change File"**
  kar dena hai.

---
