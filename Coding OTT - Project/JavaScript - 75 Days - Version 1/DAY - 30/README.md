# JS V1 - Day 30

## Task
Ek aisi website banaiye jo **Animate.css** jaisi ho, lekin uska design **modern look** me hona chahiye.

---

## Requirements
- Website me different **animation buttons/cards** available hone chahiye.
- Har animation ka naam show hona chahiye (example: Fade In, Bounce, Zoom In, Slide Left, etc.).
- User jab kisi animation par click kare, toh selected animation ka effect **live preview box** me apply hona chahiye.

---

## Animation Preview
- Ek preview section hona chahiye jisme text ya box ho.
- User jab animation button par click kare, toh preview element par animation run ho.

---

## Extra Features
- Animation repeat hone ke liye class ko remove aur re-add karna hoga.
- UI clean, responsive aur modern design me hona chahiye.

---

## Important Instruction
Project me **Animate.css CDN** use karna hai aur click event ke through animation dynamically apply karni hai.
